# Stroke Prediction — ML Project

## Overview
Predicting stroke risk using health indicators like glucose level, 
hypertension, and age.

## Dataset
Healthcare Stroke Dataset - 4909 rows, 11 features

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Key Findings
- avg_glucose_level is the strongest predictor (0.72 correlation)
- Hypertension doubles stroke risk (15% → 79%)
- Random Forest achieved best F1 score

## Libraries
pandas, numpy, matplotlib, seaborn, scikit-learn
